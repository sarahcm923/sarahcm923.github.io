Overview

We used API's in this exercise to analyze how weather patterns change as you move away from the equator. We used the OpenWeatherMap API to gather data on over 500 cities worldwide.

Once the data was collected, we used Python and Matplotlib to plot several different comparisons of weather vs. latitude, namely cloudiness, temperature, wind speed, and humidity. You can use this site to analyze the source data, visualizations and conclusions.
